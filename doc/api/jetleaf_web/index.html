jetleaf_web library

Jetleaf Web Framework Library

This library exports all core modules, annotations, context utilities, HTTP message converters, server infrastructure, routing, handlers, exception resolvers, multipart support, path matching, REST clients, template rendering, and utility classes required to build web applications using Jetleaf.

It serves as the main entry point for importing the Jetleaf web framework into your application, providing access to all features and default implementations, while allowing overrides via custom pods.
