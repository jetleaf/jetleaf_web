ReadOnlyHttpHeaders

ReadOnlyHttpHeaders class

A read-only implementation of HttpHeaders that throws UnsupportedOperationException for any mutating operations.

This class provides an immutable view of HTTP headers, useful for passing headers to components that should not modify them.
