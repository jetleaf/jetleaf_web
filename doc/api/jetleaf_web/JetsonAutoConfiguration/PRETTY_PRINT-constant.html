PRETTY_PRINT constant

String const PRETTY_PRINT

Environment property key that controls whether Jetson should pretty-print JSON output during serialization.

Property Name

jetleaf.jetson.pretty-print

Purpose

Enables human-readable JSON formatting with indentation and line breaks for easier debugging and log inspection.

Example

jetleaf.jetson.pretty-print=true

When set to true, Jetsonâ€™s ObjectMapper automatically enables `SerializationFeature.INDENT_OUTPUT`, producing neatly formatted JSON. When false or unspecified, output remains compact (no extra whitespace).

Default

false
