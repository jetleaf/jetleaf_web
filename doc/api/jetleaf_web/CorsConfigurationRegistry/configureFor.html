configureFor abstract method

void configureFor(
  1. String pathPattern,
  2. CorsConfiguration corsConfig
)

Associates the given CorsConfiguration with a path pattern.

  • pathPattern — The Ant-style pattern (e.g. /api/**, /public/*) identifying which routes the configuration applies to.
  • corsConfig — The configuration instance defining allowed origins, methods, headers, and credentials policy.

If an existing configuration is already mapped to this pattern, it will be replaced.

Example

configureFor('/secure/**', CorsConfiguration()
  ..allowedOrigins = ['https://trusted.com']
  ..allowedMethods = ['GET', 'POST']
);
