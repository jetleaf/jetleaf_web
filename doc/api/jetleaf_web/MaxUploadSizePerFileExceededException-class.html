MaxUploadSizePerFileExceededException

MaxUploadSizePerFileExceededException class

Exception thrown when a single uploaded file exceeds the per-file size limit.

This is useful for validating individual file constraints even if total upload size is within limits.

Example:

if (file.size > maxSizePerFile) {
  throw MaxUploadSizePerFileExceededException(maxSizePerFile, file.size, file.name);
}
