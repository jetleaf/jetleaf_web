Implementation of the Part interface representing a file uploaded in a multipart/form-data request.

This class encapsulates the metadata and physical file reference for an uploaded file part. It provides access to the part name, optional filename, the underlying File, and utilities for decoding content.

Responsibilities

  • Store the reference to the uploaded File.
  • Expose the part's name and original filename submitted by the client.
  • Provide decoding utilities via _encodingDecoder for reading content.

Example

final filePart = FilePart("profileImage", "avatar.png", file, decoder);
print(filePart.getName());          // Output: profileImage
print(filePart.getSubmittedFileName()); // Output: avatar.png
final file = filePart.getFile();

Creates a FilePart with the specified _name, optional _fileName, the uploaded _file, and the _encodingDecoder for content operations.
