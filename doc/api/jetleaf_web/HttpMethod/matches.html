bool matches(

Checks if this HttpMethod matches the given string, ignoring case.

This is useful when comparing an HTTP method from a request (which may be any casing, e.g., "get" or "POST") against a predefined HttpMethod constant.

Example:

final method = HttpMethod.GET;

print(method.matches('GET')); // true
print(method.matches('get')); // true
print(method.matches('POST')); // false

value - The string to compare with this HTTP method.

Returns true if the string equals this method's value (case-insensitive), otherwise returns false.

Implementation

bool matches(String value) => _value.equalsIgnoreCase(value);
