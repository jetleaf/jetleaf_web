Default implementation of the Part interface, representing a single part of a multipart/form-data request.

This class encapsulates the headers and raw content data of the part, providing access to the part's name, filename, and decoded content through an EncodingDecoder.

Responsibilities

  • Parse the Content-Disposition header to extract name and optional filename.
  • Store the raw binary data of the part.
  • Provide decoding utilities via _encodingDecoder.

Example

final part = IoPart(headers, data, decoder);
print(part.getName());
print(part.getSubmittedFileName());
final bytes = await part.getBytes();
final text = await part.getString(utf8);

Creates a IoPart by parsing the provided headers and storing the raw data. Uses _encodingDecoder to provide string decoding if needed.

The constructor automatically extracts the name and filename from the Content-Disposition header.
